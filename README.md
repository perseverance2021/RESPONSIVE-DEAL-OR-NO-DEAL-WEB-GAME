# Deal or No Deal

A browser-based Deal or No Deal game built with plain HTML, CSS, and JavaScript.

## Files

- `deal-or-no-deal.html` - main page
- `deal-or-no-deal.css` - styles
- `deal-or-no-deal.js` - game logic

## Run locally

Open `deal-or-no-deal.html` in any modern browser.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root (no folders required).
3. In repository settings, enable GitHub Pages from the main branch root.
4. Open the published Pages URL.
